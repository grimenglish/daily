일상 관리 프로그램 실행 방법

1. 압축을 풉니다.
2. 터미널 또는 명령 프롬프트를 엽니다.
3. 아래 명령어를 실행합니다.

pip install streamlit pandas openpyxl
streamlit run daily_manager_app.py

윈도우에서는 실행.bat 파일을 눌러도 됩니다.

주요 기능
- 대시보드
- 할 일 관리
- 일정 관리
- 메모장
- 엑셀 백업
- DB 백업
- 완료/반복/검색/필터/고정 메모
- 할 일 시간 알림
- 일정 시작 전/시작 시간 알림

알림 주의사항
- 앱 화면이 열려 있어야 알림이 울립니다.
- 사이드바의 '알림 자동 확인'을 켜두면 설정한 주기마다 시간이 확인됩니다.
- 소리가 안 나면 브라우저에서 페이지를 한 번 클릭한 뒤 '알림 테스트'를 눌러보세요.
